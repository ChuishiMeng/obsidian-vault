# 冯唐成事心法 - 个人行动指南

> **来源**：[[冯唐访谈-把现实当游戏才是真成事]]
> **应用场景**：健康恢复、心态调整、日常工作
> **创建日期**：2026-02-25

---

## 🎯 三个核心原则

### 1. 成事心态 > 成功心态

| 成功心态（错误） | 成事心态（正确） |
|----------------|----------------|
| "我要立刻恢复到100%" | "今天完成雾化 + 喝够8杯水" |
| "我要一个月瘦10斤" | "今天少吃一口甜食" |
| "我要成为最健康的人" | "今天比昨天好一点点" |

**行动**：每天设定 3 个能掌控的小目标，完成了就是"成事"

---

### 2. 游戏化人生

**把困难当关卡，不是灾难**

| 场景   | 游戏化思考         |
| ---- | ------------- |
| 咽炎恢复 | 当前副本，雾化是解毒药   |
| 脾胃调理 | 养成类游戏，每天积累经验值 |
| 工作压力 | 挑战关卡，打不过就换策略  |

**行动**：每天早上问自己"今天要打哪个副本？"

---

### 3. 自以为灯，自以为靠

**不外求，靠自己**

- 医生给的是拐杖，**走路靠自己**
- 别指望"贵人"来救你
- 睡好、吃好、心态好 = 真正的秘籍

**行动**：每天问"今天我为自己的健康做了什么？"
