# 第一讲 [[探究生命意义的冲动 - 古印度的精神世界]]
- 人只有体会到生命的痛苦和不安时，才可能会对内在的生命价值和意义产生认知和冲动
- 沃格林（历史哲学家）强调要用“体验”去理解古代哲人的思想，而不是用一串串复杂的概念和看似严密的理论体系，因为后者与生命体验常常是脱节和疏离的，变成一种无法贴近生命的概念游戏......他们对轴心时代的关注绝对不仅仅是“历史上发生了什么事情”，而是想要了解“那些伟大的心灵到底体验到了什么”

# 第二讲 悉达多，你为何出家？- 悉达多的苦与自由
- “方便”教化，意思是“因材施教”
- 从根本上来讲，无论社会如何进步，对于生命而言，老病死的束缚感永远围绕着我们，这也是人类永远无法停止对生命意义问题思考的根本原因。
- 从现实角度来看，我们往往在温水般的安逸中，在一切顺利的时候就失去了对“苦”的敏感。许多迫在眉睫的人生问题，其实一直在被回避和掩盖。

# 第三讲 悉达多，你觉悟到什么？ - 觉悟的内涵
- （悉达多的第一次失败尝试）“禅定”是什么？是通过一定的方法，让自己的心念安止于某个对象上。如果心念统一而且有力，就会出现与平时精神涣散状况下，完全不同的意识体验。
- 在世界上所有讲述超越体验的宗教或哲学体系中，古印度的瑜伽都算得上是主动且具有高度技术性的实践传统。这和被动等待启示降临的宗教体验形态，形成了鲜明的对比。
- 悉达多发现禅定并没有彻底解决他的烦恼......在非禅定的状态下，悉达多无法感受到解脱的喜悦，因此他认为这并不是最终的真理。
- （悉达多的第二次失败尝试）苦行也绝非解脱之道。据说他每天只吃一粒碎米，最后饿成骷髅一样，但仍然没有获得解脱。他发现，刻意虐待肉体可能也不是真正的解脱之道。
- （悉达多的成功尝试）悉达多发现，之所以有生老病死，本质上是因为“有生”。但究竟什么叫“生”？其实，我们无法为“生”找到一个绝对的时空坐标。我们只不过想当然地认为在某个时刻，生命开始了。而且从那以后，我们在成长过程中不断地强化这样一个看法。貌似我作为一个生命个体，是一种永恒的存在。即便我们也知道，或者说我们也同时知道有衰老与死亡
- 这种“自我感”其实是人类根深蒂固的一种错觉。在悉达多看来，“有”不过是执取的结果而已，本就不存在一个本体的“物”，而是靠强大幻觉支撑起来的。这种认知让我们误以为有一个“我”存在。
- 为何有这种认知的执取？他发现，其实是心中的贪爱在推动这种“执取心”的运作。原因在于，一旦我们对事物产生“好”与“坏”的判断后，就会对所喜爱的产生“占有欲”，对所讨厌的感到“厌恶感”。无论是想要占有还是百般厌恶，这都属于“贪爱”的范畴。
- 悉达多的觉悟就是要回答这样一个核心问题：人为何会恐惧死亡？而他的最终答案是：我们错误地理解了“我”。
- 我所贪求的对象，最终界定了我的本质。只要我有贪求的对象，我就会产生牢固的执着。
- 我们因为贪求外在的事物，从而产生了一种对生命的错误理解：因为执着于有一个“恒常的我”存在，于是不断地索取和害怕失去，导致无尽的烦恼和焦灼。而悉达多在菩提树下的觉悟，正是看清了这一点，也因此获得精神的解脱。  
# 第四讲 为何“生”也是人间之苦？
- 四圣谛，是佛陀所觉悟的四个至高无上的道理：
	1. 苦：是指世间一切皆苦
	2. 集：是指造成苦的原因
	3. 灭：是指灭苦的结果
	4. 道：是灭苦的方法
- 人生八苦：包括生、老、病、死、忧悲恼、怨憎会、恩爱别离、所欲不得。
- "求不得苦"的范围，远不能被那些我们用钱购买的商品所囊括。它还包括对生活中的一切事物的某种欲求：
	1. 恋人想要控制对方，却又无能为力
	2. 外出旅行想要风和日丽，天气却不能如愿
	3. 努力工作，却达不到理想回报
以这样的角度去观察，我们会发现这种苦贯穿了我们的日常生活，成为生命的常态。只不过我们总是在这些求不得苦的缝隙间，用尽全力去捕捉一个个小确幸。

# 第五讲：我们为什么会苦
- 五蕴（色蕴、受蕴、想蕴、行蕴、识蕴）。“蕴”有集合的意思。
- 色蕴
	- “五根”：眼，耳，鼻，舌，身（意）
	- “五尘”，包括色、声、香、味、触，是五根所感受到的外在对象
- 受蕴：是认知的最初结果，包括三种感受：
	1. 苦受
	2. 乐受
	3. 不苦不乐受
- 想蕴：是心开始对感知的对象进行初步分辨
- 行蕴：是潜在的意志，推动身心的行动
- 识蕴：由意识来决定识别的对象是什么物体、有什么功能，并进一步分别好坏，产生后续的心理反应和行动决策
- 佛教讲“一切皆苦”，是说我们处在一种“跷跷板式”的、永远不满足、不稳定的状态中。而造成这种“苦”的原因，也并非我们无法达到所追求的目标，而是一旦达成暂时的目标，就会陷入新的、不满足的漩涡当中，永无休止。
- 我们之所以感到苦，并不是取决于你所遇到的人是谁，而是我们对这些情境的认知。
- 苦的真正原因不在于所谓客观的对象，而在于我们的执着。因为我们所贪取的对象，其实不可能被我们真正掌控：
	1. 一切事物都在无常变化
	2. 就算我们好像得到了所贪之物，但转瞬间就已经失去了它
	3. 在它变化的过程中，我们的感受也会随之变化
- 苦的原因，其实是无明所带来的贪爱心，是“八苦”的根本原因。无明就是对真理无知的意思。

# 第六讲：从“苦”走向“佛系”
- 贪与嗔、追求与放弃，构成了二元对立。我们想要获得幸福，便开始穷追不舍；一旦心力交瘁、失去希望，又开始否定目标，认为人生不过是一场虚无的游戏。
- 这些观念逻辑都不是佛教意义上的“灭苦之道”。在佛教的视野里，解决人生之苦的核心在于执着。而要做到不执着，就要跳脱出非此即彼的二元逻辑。
- 不执着的人生状态，其实就是不为任何一种幸福的标准所固化，也不必设定某种非它不可的结果。我们反而在生命的每一刻，都能体会当下的无限开放和自由。在禅师的眼里，这其实就是觉悟的生命。

# 第七讲：你为何如此执着？
- 贪，其实就是对自己喜欢的东西执着不舍、念念不忘。所谓的“拥有”，其实是一种认知错觉。因缘的不断变化，很快会展现出万物不随个人意愿而改变的真相。最终，只不过是看上去拥有而已。在时时刻刻都在变化的世间万物中，我们根本找不到真正拥有的那一刻。
- 嗔，是人因为讨厌或拒绝而产生的嗔恨心。嗔的根源表面上是环境的问题，是外在的境界让人产生烦恼。但仔细想想，不过是因为境界不如人意，加之内心深处的无知在起作用。试图让一切受我主宰，以求身心的欲望满足，因此一旦不能达到，心中就会产生种种不乐意的情绪，也就是“嗔”。
- 慢，等同于傲慢的意思，是想要压倒他人的自大虚荣之心。
- 疑，主要是指不相信因果业报及缘起空的道理，因此无法得到解脱。
- 痴，约等于无知或无明，描述人心的一种暗昧不清的状态，表现为对世俗的是非善恶无法判断，对世界的真相也无法了解。

# 第8讲：世界是常还是无常？
- 仔细想一想，难道我们的生活真的是重复如一的吗？就以刷牙漱口来说，每次刷牙的动作难道是完全一样的？显然不是。每次挤牙膏的动作和牙膏用量的多少，其实都有所不同。又比如搭乘地铁和公交，看上去周而复始，但每天碰到的人都各有不同。就算是同一行程的旅伴，他们每天的装扮和精神状态也都是不同的。
- 那么，表面上看不到变化的生活，当真是不变而乏味的吗？殊不知，那只是因为我们的心动荡且粗糙，无法体验到表面稳定的生活底下所泛起的涟漪。
- 只有我们的心变得专注和细腻，才能观察到更多的无常。可一旦心变得粗糙与混沌，就像镜头失去了焦距，无法看见细节的变化，当然会认为生活单调而重复。

# 第九讲：我真的存在吗？
- 佛学中有一个专门的术语叫“虚空”，意思是想要认知一个事物的本质，可以从空间、时间等角度来进行分析。最终发现在每个组成部分中，都无法找到“我”的本质，由此推出了“无我”的结论。
- 我们在尘世中走来走去，忙来忙去，每一个时空的碎片好像是你，又不是真的你。你找不到那一个单一的、不变的、本质的你：15 岁的你是你， 40 岁的你也是你，请问他们是同一个人吗？既是，又不是。
- 我其实就如同一条河流，看上去在连续不断地变化，但仍然维持着一种可识别的轮廓。它有自己流动的方向与形态，但你却不能说哪一滴水就是这条河流，你也不能说前一秒的那朵浪花就是这条河流。你会感觉到过去、现在、未来奔流的每一滴水珠，每一朵浪花，乃至每一个漩涡都是这条河流，但同时又都不是这条河流。
- 在责骂发生时的我，其实早已经过了时空转化和无常变动。我的眼、耳、鼻、舌、身早已远离上午的境界。那为什么现在的我仍为上午的事情焦虑不安呢？这不过是因为你的意识里预设了一个不变的我，所以上午感到沮丧的那个我，被硬生生地带到了中午、下午，乃至第二天。

# 第十二节：八正道与定的修行
- 慈悲与智慧：一体两面

佛教中的慈悲心，其实是建立在理解他人、同情他人的基础之上的，而不是以自我中心为前提。

从这一点出发，也就能更好地理解和趋向于“无我”。所以慈悲和智慧对于佛学而言，其实是一体两面：
1. 越能理解他人的苦，也就越容易生起慈悲心。
2. 自然也就弱化了执着。

因此，不仅不会沉浸在“我”的痛苦之中、怨天尤人，反而会激励自己更清楚地认识“无我”的真相。在处理自身烦恼的同时，也能更加积极地帮助他人。

- 四圣谛：
	- 苦
	- 集：贪嗔痴慢疑
	- 灭：诸行无常，诸法无我，涅槃寂静（烦恼的止息）
	- 道：戒（正语、正业、正命）、定（正念、正定、正精进）、慧（正见、正思维）

# 第十三讲：“四念处”的内涵与实践
- 观身不净，用这种思维观察的方法，来对治我们对色身的贪欲。
- 观受是苦，观察到我们的苦和乐，不过是一体两面，不过是无常变动的身心现象。
由于我们的偏好与执着，会产生一种要么沉溺、要么沮丧的心理模式。我们总是始于执着于乐，终于苦中。
- 观心无常，核心就是观察念头的变化。
- 观法无我。一切法中都找不到一个恒常不变的我，自然也就明白了“无我”。

# 第十四讲：禅修
- 观呼吸。以呼吸作为觉察的对象，请务必保持放松自然的呼吸。不要刻意去用腹式呼吸，或者刻意引导呼吸到某个部位，这些都属于控制呼吸。
- 禅修并不像很多人想象的那样。我们可以盘腿坐着，其实它不限于任何形式。

如果你掌握了方法的要领，走路、合掌也同样可以禅修。但对于初学者来说，之所以要用静态的打坐来练，是因为：
1. 当我们处于动态时，心会随着眼、耳、鼻、舌、身等五根向外攀援
2. 当处于静态时，我们强制性地隔绝了大部分干扰，能更好地往内专注，心也更有力量

# 第十五讲 缘起
- 缘起法想告诉我们的是：实现理想固然需要我们的努力，但我也只能尽力让条件成熟，无法控制结果。而“我执”（也就是想要控制结果的念头）就恰恰违背了缘起法。

# 第十六讲 业力
- 业力的定义：它是用来解释生命流转的动力及作用。

佛教所谈的“业”，是指人的行为、语言和心理活动会留下生命的印记和惯性，并产生后续身心造作的动力。
- 人更多以善心去生活时，至少会感受到身心的愉悦与安康。

就像很多心胸坦荡的人，尽管一辈子饱受挫折，但在晚年里活得宽容自在。

# 第十七讲：从缘起看现代人的孤独感
- 佛教中的慈悲，并非是主体对客体的施舍与怜悯。而是感受到身处苍茫无明的世界中，我们彼此连接、互相作用，并且依着因缘，互相成就着对方。

- 佛教中慈悲的不同之处，关键在于“无我”和“无所得”。也就是说：
1. 我帮对方时，并不是带着一个“我”来的。
2. 最终也并非要由“我”得到回报。

- 既然每个人的当下，都依着各种我们无法主宰的因缘条件而成，那么就不要简单粗暴地否定对方。即使当下他们的选择在你看来不够合理，也无需因此产生强烈的我执烦恼。

- 既然我们在此刻进入到对方的生命中，一方面要尽自己的智慧和力量去帮助对方，但也要深知背后因缘的错综复杂。对于任何事，非一己之力就能在当下成功。

# 第十八讲：什么是空？
- “空”的哲学化的表达就是“无自性”，也就是无本质的意思。

# 第十九讲：善恶是绝对的吗？
- 世界上的一切，本来都不是我们所想象的那样。不存在一个真实事物的生和灭，而只是如梦如幻的生和灭，这才是不生不灭的真正含义。
- 《佛学》意义上的不垢不净、不善不恶，就是在提醒我们不要陷入二元化的对立中。

对于社会中的恶，我们可以批评，但要警惕批评背后带着偏执。因为这种偏执会同样引导我们，最终变成自己所仇恨的样子。

# 第二十讲：以手指月：真理能否言说？
- 禅宗有“手指月”的比喻。是说当手指指向月亮时，我们应该顺着手指去看月亮，而不是盯着手指看。
- 真理是否根本上就无法被用概念表达出来呢？

# 第二十一讲 无自性：如何正确理解“空”？
- 从现象来看，我们对他人和世界有一种想要占有、控制的冲动，这使得我们无法摆脱得失之苦。这种本能的贪执，佛教认为来自根本的无知，也就是“无明”。
- “无明”就是认为一切事物都是有本质的，无论是外界的各种事物还是对“我”的认知，都会对其浮现一种“真实存在”的认知影像，这在佛学中被称为“自性见”。
- “自性”有三个特征：自有、独一和常住。“自有”就是认为自身具备真实的本体；“独一”则认为事物可以单独存在，不依赖任何其他条件；而“常住”则是指事物没有时间维度上的变化。
- 恰恰相反，将“空”当作一种绝对的“无”，反而违背了“空”的意涵，因为“空”的核心是“无本质”，也就是“无自性”，而非“不存在”。
- 这本书记到这里，也进入了最关键的阶段，而在我看来，佛学所具备的解忧效用的前提，恰恰是你正确地理解了“空”，只有如此，才能显发出它的独特力量。而如果你的认知还停留在习以为常的二元论思维当中，就算道理再通俗，言语再温柔，都可能只是暂时的“止痛药”，无法真正解决烦恼和问题。

# 第二十二讲 龙树菩萨中论，八不偈
- 不生亦不灭，不常亦不断，不一亦不异，不来亦不出（去）。
- 为何《中论》一开始反而否定了有本质的生灭、常断、一异和来去的存在呢？
- 前一秒的我和后一秒的我到底是什么关系？如果“生”真有其本质可言的话，那么就不应该会有变化，那又怎么可能有“灭”呢？所以按照佛教的看法，“不生不灭”其实是说，任何事物的存在都不是实有的，只是随着因缘变化而已。所以生命从一开始就不是“无中生有”，生命到了终点也并非掉进了黑洞。

# 第二十四讲 金刚经
- 有僧人问大珠惠海禅师：和尚修道，还用功否？
- 曰：用功。
- 曰：如何用功？
- 曰：饥来吃饭，困来即眠。
- 曰：一切人总如是，同师用功否？
- 曰：不同。
- 曰：何故不同？
- 曰：他吃饭时不肯吃饭，百种须索。睡时不肯睡，千般计较。所以不同也。

# 第二十五讲 我相、人相、众生相、寿者相
- 但是一旦对方说的话、做的事触犯到你的敏感点——无论是损害了你的利益，还是和你的认知、观念有了冲突——是不是常常会产生强烈的对立感？这就是某种意义上的“我相”。
- 接下来，对方的形象在你心里挥之不去，走路想，吃饭想，晚上该睡觉了还在愤愤不平，这就是“人相”，即他者之相。
- 但是仔细想一想，你心里想的这个人就是触怒你的那个人吗？那一瞬间早已过去，他还是那个他吗？从最简单的“无常”道理就可以推出，当然不是，但是你在脑海里构造出的鲜明的“人相”，已经变得牢不可破。
- 当然，这个固定的、本质化的人相，其实是被你的意识建构起来的，这来自“我相”所带来的与他人的对立，“我相”越坚固，“人相”自然也就越难以消除。
- 现代社会的标准化和科层化，使每个人都被赋予某个模式化的“我相”要求。这当然会让我们感觉受到了极大的束缚，于是就需要在某些场合放飞自我，让自己得到极大的发泄，比如一些年轻人对极限运动的追求，对迷笛音乐节的迷恋，其实都是对日常生活中程式化的“我相”的各种抵抗。
- 而其中最为核心的就是“无我相”，因为有了“我相”，“人相、众生相、寿者相”就会自然显现出来。
