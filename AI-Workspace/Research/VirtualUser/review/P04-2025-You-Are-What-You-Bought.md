## 一、研究背景

### 1.1 问题场景（一句话说清楚）

**核心问题**：如何从用户购买历史自动生成用户 Persona 标签？

**通俗理解**：

```
场景：电商平台想给用户打标签

传统做法：
  人工标注 → 成本高、质量不一

本文做法：
  购买历史 → LLM 生成标签 → 主动学习优化
  成本降低 90%
```

---

## 二、核心贡献

### 2.1 一句话说清楚

**用 LLM 从电商购买历史生成 Persona 标签，通过主动学习 + 近似去重降低成本 90%。**

### 2.2 核心工作

**行为抽象**：

```
输入：购买序列 [商品1, 商品2, ...]
输出：Persona 短语 ["砍价达人", "品质控", "冲动消费"]
```

**成本优化**：

| 方法 | 成本 |
|------|------|
| 全量标注 | 100% |
| + 主动学习 | 30% |
| + 近似去重 | 10% |
