# Inferring Air Quality for Station Location Recommendation

**会议**: ACM KDD 2015  
**作者**: Hsun-Ping Hsieh, Shou-De Lin, Yu Zheng  
**机构**: National Taiwan University, Microsoft Research

## 核心贡献

解决空气质量监测的两个关键问题：
1. **推断问题**: 如何从稀疏监测站数据推断任意位置的实时空气质量
2. **选址问题**: 如何选择新建监测站的最佳位置

## 挑战

- **数据稀疏**: >99%的位置没有空气质量数据
- **空间异质**: 不同位置空气质量差异大
- **时间动态**: 空气质量随时变化

## 方法论

### 1. 半监督推断模型

**数据融合**:
- 气象数据（风速、湿度、温度）
- 人群移动数据
- 道路网络结构
- 兴趣点（POI）分布

**模型设计**:
- 利用有标签数据（监测站）+ 无标签数据（其他位置）
- 图模型建模空间相关性
- 时序模型捕捉时间依赖

### 2. 熵最小化选址模型

**目标**: 选择新建监测站位置，最大化信息增益

**方法**:
- 计算每个候选位置的不确定性（熵）
- 选择能最大程度降低整体不确定性的位置
- 考虑预算约束

## 关键技术

### 异构数据融合
```
气象 + 移动 + 道路 + POI → 统一特征空间
```

### 图模型
- 节点：空间位置
- 边：空间相关性
- 权重：相似度

### 半监督学习
- 标签传播
- 协同训练
- 图正则化

## 实验验证

- **数据**: 北京空气质量数据
- **对比方法**: 一系列主流方法
- **结果**: 显著优于传统方法

## 与U-Air的关系

- **U-Air (KDD 2013)**: 空气质量推断
- **本论文**: 增加监测站选址功能
- **共同点**: 都使用异构城市数据

## 应用价值

1. **实时监测**: 获取全城空气质量地图
2. **规划决策**: 指导新监测站建设
3. **公共健康**: 提供精细化的健康建议
4. **资源配置**: 优化监测网络布局

## 创新点

1. **半监督框架**: 解决数据稀疏问题
2. **异构数据融合**: 整合多源城市数据
3. **主动学习**: 熵最小化选址策略
4. **实用价值**: 直接指导政府决策

## 技术亮点

### 不确定性量化
- 预测结果 + 置信度
- 识别高不确定性区域
- 指导监测站部署

### 主动学习循环
```
推断 → 识别高不确定性区域 → 建议新站点 → 更新模型 → ...
```

## 启发

- **数据稀疏是常态**: 需要半监督方法
- **异构数据互补**: 不同数据源提供不同信息
- **主动学习价值**: 智能选择最有价值的标注
- **城市规划应用**: 数据驱动的基础设施选址
