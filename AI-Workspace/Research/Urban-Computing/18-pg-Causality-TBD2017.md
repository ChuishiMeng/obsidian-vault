# pg-Causality: Identifying Spatiotemporal Causal Pathways

**期刊**: IEEE TBD (Transactions on Big Data) 2017  
**作者**: Julie Yixuan Zhu, Chao Zhang, Huichu Zhang, Shi Zhi, Victor O.K. Li, Jiawei Han, Yu Zheng  
**机构**: HKU, UIUC, Microsoft Research

## 核心贡献

识别空气污染物的时空因果路径：
1. **理解污染传播**: 不同污染物如何积累和传播
2. **支持政策制定**: 为污染治理提供科学依据

## 挑战

1. **数据噪声**: 原始数据包含大量低污染期和噪声
2. **计算复杂**: 大规模时空数据的因果分析复杂度高
3. **路径复杂**: 多污染物交互 + 环境因素影响

## 方法论

### pg-Causality: 模式辅助因果分析

结合 **模式挖掘** + **贝叶斯学习**

### 1. 模式挖掘阶段
- **目标**: 抑制噪声，减少复杂度
- **方法**: 捕捉每个传感器的频繁演化模式(FEPs)
- **筛选**: 选择模式匹配的传感器作为"因果源"

### 2. 贝叶斯学习阶段
- **模型**: 高斯贝叶斯网络(GBN)
- **编码**: 局部因果 + 时空因果
- **整合**: 环境因素（气象）减少偏差

## 关键技术

### 频繁演化模式(FEP)
- 捕捉传感器值的时序变化规律
- 过滤随机噪声
- 保留有意义的因果信号

### 高斯贝叶斯网络(GBN)
```
[传感器A_t] → [传感器B_{t+Δ}] 
       ↑              ↑
    [气象因素]   [气象因素]
```

### 时空因果图
- **节点**: 传感器 × 时间
- **边**: 因果关系
- **权重**: 因果强度

## 实验验证

- **数据规模**: 982个空气质量传感器
- **时间范围**: 2013-06-01 至 2015-12-19
- **区域**: 中国三个地区
- **对比方法**: 传统因果结构学习方法
- **优势**: 时间效率 + 推断准确性 + 可解释性

## 应用价值

1. **污染溯源**: 找到污染源头
2. **传播预测**: 预测污染扩散路径
3. **政策指导**: 针对性治理措施
4. **科学研究**: 理解污染形成机制

## 创新点

1. **模式辅助**: 用模式挖掘降低噪声和复杂度
2. **贝叶斯建模**: 统一框架处理时空因果
3. **环境整合**: 考虑气象因素避免虚假因果
4. **可扩展**: 处理近千个传感器

## 技术亮点

### 两阶段设计
```
模式挖掘 → 候选筛选 → 贝叶斯学习 → 因果图
```

优势：
- 降低计算复杂度
- 提高结果可靠性
- 增强可解释性

### 时空联合建模
- 空间维度：传感器间因果关系
- 时间维度：时滞因果关系
- 联合优化

## 启发

- **模式 + 统计**: 结合领域知识的机器学习
- **时空因果**: 城市数据的特殊挑战
