### 2.1 数据采集增强

#### 2.1.1 稀疏数据增强

**Urban Computing挑战**：
- 空气质量监测站稀疏（>99%位置无数据）
- 轨迹数据低采样率（间隔2-5分钟）
- 部分区域/时段数据缺失

**虚拟用户贡献**：

| 数据类型 | 传统方法 | 虚拟用户增强 |
|----------|----------|--------------|
| 空气质量 | 仅靠监测站 | 虚拟用户"感知"报告 |
| 人流数据 | 传感器采集 | 虚拟用户轨迹生成 |
| 交通状态 | GPS轨迹 | 虚拟乘客体验报告 |

**技术方案**：
```python
# 虚拟用户增强的空气质量感知
class VirtualUserAirSensing:
    def __init__(self, location, profile):
        self.location = location
        self.profile = profile  # 敏感度、健康状况等
    
    def perceive_air_quality(self, pm25_value):
        """将客观数值转化为虚拟用户主观感知"""
        if self.profile.sensitivity == "high":
            if pm25_value > 50:
                return "空气浑浊，不适合户外运动"
        else:
            if pm25_value > 100:
                return "空气不好"
        return "空气还可以"
    
    def report(self):
        """生成虚拟用户感知报告"""
        return {
            "location": self.location,
            "objective_value": get_sensor_value(self.location),
            "subjective_report": self.perceive_air_quality(...),
            "behavior_change": self.adjust_behavior(...)
        }
```

**启发来源**：

**核心思路**：
```
真实用户行为日志 → 行为模式学习 → 虚拟用户行为生成
```

**多维度行为建模**：

| 维度 | 真实数据源 | 虚拟用户属性 | 建模方法 |
|------|------------|--------------|----------|
| **时空行为** | GPS轨迹 | 移动轨迹、活动范围 | 轨迹生成模型 |
| **功能使用** | 签到/POI | 访问的功能模块 | 序列模式挖掘 |
| **时间规律** | 时间戳 | 日/周/季节模式 | 周期性建模 |
| **社交网络** | 社交关系 | 虚拟社交连接 | 图生成模型 |

**行为生成框架**：
```python
class VirtualUserBehaviorGenerator:
    def __init__(self, profile, urban_environment):
        self.profile = profile  # 年龄、职业、收入、偏好
        self.environment = urban_environment  # POI、路网、功能区
    
    def generate_daily_trajectory(self, date):
        """生成一天的轨迹"""
        # 1. 确定主要活动（基于功能区）
        home = self.assign_home_location()
        work = self.assign_work_location()
        
        # 2. 生成时间表（基于行为模式）
        schedule = self.generate_schedule(date)
        
        # 3. 轨迹生成（基于T-Drive方法）
        trajectory = []
        for activity in schedule:
            path = self.generate_path(
                trajectory[-1].location if trajectory else home,
                activity.location,
                activity.start_time
            )
            trajectory.extend(path)
        
        return trajectory
    
    def generate_path(self, start, end, time):
        """生成两点之间的路径（类似T-Drive）"""
        # 基于时间依赖的地标图
        landmark_graph = self.get_time_dependent_graph(time)
        return self.find_route(start, end, landmark_graph)
```

class EmergencySimulation:
    def __init__(self, virtual_population, city_model):
        self.population = virtual_population
        self.city = city_model
    
    def simulate_evacuation(self, hazard_location, hazard_type):
        """模拟应急疏散"""
        # 1. 影响范围评估（类似Black Hole检测）
        affected_area = self.assess_impact(hazard_location, hazard_type)
        
        # 2. 虚拟用户响应
        evacuation_requests = []

        return simulation_results
