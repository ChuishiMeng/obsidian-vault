# Knowledge Base Construction 论文笔记

**标题**: Knowledge Base Construction for Knowledge-Augmented Text-to-SQL  
**发表**: ACL 2025 Findings

---

## 1. 知识库构建方法 (KAT-SQL)

### 两阶段框架

**Phase 1 - 知识库构建**：
- 从现有数据集收集知识
- 使用 LLM 自动扩展生成新知识条目

**Phase 2 - 知识增强 SQL 生成**：
- 检索相关知识 → 精炼适配当前查询 → 生成 SQL

### 关键技术

- 使用 few-shot 示例引导 LLM 生成
- 采样和排列策略增加知识多样性
- 对比学习训练检索器提高准确率

---

## 2. 知识表示方法

**格式**：简洁的自然语言句子

**示例**：
```
"Abnormal white blood cell count refers to WBC ≤ 3.5 or WBC ≥ 9.0"
```

**特点**：
- 直接关联数据库 schema 列名
- 可跨查询/跨数据库复用

## 3. 与本研究的关系

### 核心映射

| 论文方法 | 我们的业务消歧 |
|---------|--------------|
| 领域术语解释 | 业务概念定义 |
| 数据库 schema 映射 | 概念-指标-维度映射 |
| 知识条目 | 消歧规则/案例 |
| 知识库检索 | 概念相似度匹配 |

### 可借鉴方案

1. 从历史问答日志构建"问题-概念-SQL"三元组知识库
2. 采用检索-精炼-生成三阶段消歧流程
3. 使用结构化自然语言表示业务概念
