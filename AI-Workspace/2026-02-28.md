#### 为什么要使用本地模型？
- **无限 tokens**：无速率限制，24/7 使用

- **零 API 费用**：只需电费

#### Alex 的实践案例
Alex 用 3 台 Mac Studio 运行本地模型（Qwen 3.5），实现了"AI SaaS 工厂"：
- 4 个 OpenClaw agent 协同工作
- 1 个 agent (Ralph) 负责 QA
- 闭环自我改进系统
- 完全免费（只用电力）

#### 如何在 Mac Mini 上设置
**硬件要求**：
- 至少 32GB 内存的 Mac Mini（Qwen 3.5 需要 20GB，留点余量）
- 如果只有 16GB 基础款，可以用更小的模型

**设置步骤**：
1. 下载 LM Studio — lmstudio.ai（免费）
2. 搜索 Qwen3.5-35B-A3B-4bit（在 Discover 标签）
3. 下载（约 20GB）
4. 加载模型
5. 让 OpenClaw 连接到它

#### 使用策略：混合模型
- **前沿模型（Anthropic/ChatGPT）作为"大脑"**：负责规划
- **本地模型作为"肌肉"**：负责执行
- 执行占 90% 的 token 使用量，这样可以大幅节省成本
