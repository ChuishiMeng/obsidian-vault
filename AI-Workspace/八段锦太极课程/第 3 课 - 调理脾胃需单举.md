## 📝 课程摘要

### 核心功效

**筋骨拉伸**：
- 拉伸脊柱间小肌肉群
- 增强脊柱灵活性与稳定性
- 防范和治疗脊柱及肩颈疾病

**经络拉伸**：
- **胃经**：改善消化不良
- **脾经**：健脾去湿
- **肝经**（最重要）：疏肝解郁

### 为什么名为"调理脾胃"却主要拉伸肝经？

《黄帝内经》讲到：
- **肝藏血，血主百脉**
- **肝瘀是很多病症的内因**，其中包括脾虚
- **调理脾胃，健脾去湿，必先疏肝**
- 肝经之火不平，则不能达到健脾的目的

### 健康收益

- 促进肠胃蠕动
- 提高消化代谢能力
- 改善肥胖、便秘、长斑
- 收获健康 + 美丽

## 🎯 动作要领

### 起势（同第 2 段）

1. 双手展平，吸气向上至中丹田
2. 手扇形打开，掌根相对
3. 三贴合：掌根贴→手臂贴→身体贴合
4. 向外展臂，气吸满，屏住呼吸

### 主体动作

| 步骤 | 动作 | 呼吸 |
|------|------|------|
| 1 | 左手向下，右手上托（上托下按） | 屏息 |
| 2 | **转身** | 呼气 |
| 3 | 手臂向下落（贴合身体） | - |
| 4 | 后手**旋掌**，从后边向上起 | - |
| 5 | 回身转正 | 吸气 |
| 6 | 重复 3 次后换方向 | - |

### 关键要点

1. **掌根紧锁**：手腕往回勾，锁住手掌（第 3-8 段通用）
2. **手臂伸直**：不能弯曲，拉伸阳经
3. **落手贴身体**：下落的手尽可能贴合身体
4. **起手远离身体**：上起的手尽可能远离身体
5. **旋掌**：掌根朝正后方，从后边起

## ⚠️ 常见错误

| 错误 | 正确做法 |
|------|----------|
| 手臂弯曲 | 保持手臂伸直，拉伸阳经 |
| 手腕松开 | 掌根紧锁，往回勾 |
| 落手不贴身体 | 下落时尽可能贴合身体 |
| 起手不离身体 | 上起时尽可能远离身体 |
| 旋掌方向不对 | 掌根朝正后方 |
