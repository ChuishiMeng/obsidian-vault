## 📝 课程摘要

### 静站桩 vs 动站桩

| 类型 | 功效 |
|------|------|
| **静站桩** | 倾向于气的培补，让体内能量充盈 |
| **动站桩** | 让能量流动起来，运输到每一个神经末梢，保持循环顺畅、代谢正常 |

## 🎯 动站桩练习流程

### 准备：混元桩（3 分钟）

**步伐**：
- 动站桩的步伐要大于混元桩
- 约一个半肩宽（可以大一点）

**复习四个方法**：
1. 虚灵顶劲
2. 含胸拔背
3. 沉肩坠肘
4. 松腰沉胯

**复习三个观察**：
1. **观身**：身体有没有前倾、后仰、左歪、右斜（八面支撑）
2. **观呼吸**：有没有憋气、急促、不畅
3. **观心神**：念头有没有飞出去，是否处在当下

## 🎯 第一桩：妙转太极桩

### 功效
- **培补肾气**
- **提升精气神**

### 动作要领

**起势**：
1. 左手回到丹田，但不贴住身体（保持一拳距离）
2. 右手下降到与左手同样高度
3. 形成阴阳鱼的一面
4. 左手到右手之间有一个"口"（缺口）

**动作**：
1. **转体**：腿不要动，直接旋转腰
2. **转不动时**：两个手交换位置
   - 右手收回下丹田
   - 左手到外边去
3. 形成左式的阴阳鱼
4. 再向右转身
5. 转不动时，手再交换位置

**要点**：
- 身体不能歪、不能斜
- 保持身体直立
- 先旋转身体，再动手
- 手臂与身体保持一定距离，不要贴住
- 腿部要求非常大，可能会很僵硬，要坚持

### 练习次数
- **9 次为一组**
- **做 2 组**

## 🎯 第二桩：旋转乾坤桩

### 功效
- **运转丹田之气**
- **让五脏六腑得到充分滋养**

### 动作要领

**起势**：
1. 身体站直
2. 双手慢慢打开
3. 右手在上，左手在下
4. 抱一个球（两手之间是下丹田）

**动作**：
1. 两掌心交换位置
2. 身体完全不动
3. 回到站桩姿势
4. 再继续
5. 掌心相对，不断循环

**口诀**：
- **无形、无相、无棱、无角**
- 放松做
- 闭上眼睛感受
- 动作不宜太快，慢慢做
- 保持均匀速度，不能忽快忽慢
- 想象这里有一个球，围绕着球的 360 度抚摸

**要点**：
- 手臂动作不大，只有小手臂和手腕动
- 舌抵上颚
- 眼睛微闭
- 外导内引：外旋转的是手，内导引的是丹田之气

### 练习次数
- **9 次为一组**
- **做 3 组**

---

## 🧘 呼吸配合

- 保持均匀呼吸
- 不要干预呼吸
- 外导内引，丹田气充盈时会有肠鸣（肚子里咕噜咕噜叫）

---

## ⚠️ 关键要点

1. **混元桩准备**：先站 3 分钟混元桩，把周身气机生发起来
2. **身体中正**：八面支撑，不前倾、不后仰、不左歪、不右斜
3. **三个观察**：时刻做自己的观察者
4. **妙转太极桩**：先转体，转不动再动手交换位置
5. **旋转乾坤桩**：身体不动，只有手旋转
6. **均匀速度**：不能忽快忽慢
7. **腿部坚持**：腿部可能会酸麻胀，是僵硬的感觉，坚持一下

---

## ⚠️ 常见错误

| 错误 | 正确做法 |
|------|----------|
| 身体歪斜 | 保持八面支撑，身体中正 |
| 手臂贴身体 | 保持一拳距离 |
| 动作太快 | 慢慢做，保持均匀速度 |
| 呼吸急促 | 保持呼吸顺畅，不要干预 |
| 心神飞出去 | 及时拉回当下 |
| 想象太多 | 无形无相，不要想象，跟随动作感受 |

## 📋 练习建议

1. **练习顺序**：混元桩（3 分钟）→妙转太极桩（2 组）→旋转乾坤桩（3 组）
2. **步伐**：动站桩步伐大于静站桩（一个半肩宽）
3. **三个观察**：时刻做自己的观察者
4. **均匀速度**：不能忽快忽慢
5. **腿部坚持**：可能会出现酸麻胀，坚持一下
