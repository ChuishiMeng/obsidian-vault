## 🎯 动作要领

### 起势（第 2-8 段通用）

1. 双手展平，吸气向上至中丹田
2. 手扇形打开，掌根相对
3. **三贴合**：掌根贴→手臂贴→身体贴合（背部微圆）
4. 向外推肩展臂，指尖向外，双臂伸直

### 主体动作

| 步骤 | 动作 | 呼吸 |
|------|------|------|
| 1 | 虚膝下蹲，重心转移 | 呼气 |
| 2 | 提膝跨开，落马步 | - |
| 3 | 摆脚蹬腿，转弓步 | 呼气 |
| 4 | 后手收回耳朵，前手伸直 | 吸气 |
| 5 | 钻空心拳，向两端使劲 | - |
| 6 | 收脚跟，扣脚掌，回马步 | - |
| 7 | 穿掌（前手从后手上方穿出） | 呼气 |
| 8 | 重复 3 次后换方向 | - |

### 关键要点

1. **空心拳**：手心打空，约一拳大小
2. **五指张开**：尤其大拇指和食指要尽量张开
3. **内扣手腕**：开弓时手腕内勾（核心动作）
4. **向两端使劲**：肘和掌根同时用力
5. **身体中正**：马步时不要前俯后仰
6. **手臂不动**：马步转弓步时，先摆脚蹬腿，手臂后动

## 🧘 呼吸配合

| 动作 | 呼吸 |
|------|------|
| 开弓（拉弓） | 吸气 |
| 穿掌（收弓） | 呼气 |

**口诀**：开弓没有回头箭——开弓后一定要穿掌把气呼出去

---

## ⚠️ 常见错误

| 错误 | 正确做法 |
|------|----------|
| 手掌跑到身体前面 | 向两端使劲，肘和掌根同时用力 |
| 手腕不内扣 | 开弓时手腕内勾 |
| 五指并拢 | 五指张开，尤其大拇指和食指 |
| 马步过低导致僵硬 | 半蹲即可，保持身体放松中正 |
| 重心压在脚尖 | 小腿保持直立，前后脚均匀受力 |

1. **每次练习**：左右各 3 次开弓
