## 📝 课程摘要

### 站桩的核心价值

**补充元气最好的方法**：
- 元气是气血运行的主要动力
- 元气充足 → 气血循环顺畅 → 排毒代谢功能恢复正常
- 对消化不良、便秘、长斑、长痘有较好改善

**修炼心的功法**：
- 很多同修有失眠多梦、焦躁抑郁的情况
- **身不正则气不通，气不通则意不尽，意不尽则神不安**
- 站桩通过中正骨骼，让我们回到身正的状态
- 这是一个修身养性的方法

### 站桩分类

- **静站桩**：本节课学习的类型
- **动站桩**：后续课程

## 🎯 两个静站桩

### 第一桩：吐故纳新桩

**功效**：
- 把身体和呼吸完全调平
- 身体紧张会带来呼吸不顺畅
- 通过身体放松 + 吐故纳新练习
- 达到通透放松的境界

**动作要领**：
1. 虚膝下沉
2. 重心向右移动
3. 左脚平开（与肩同宽）
4. 双手环抱下沉
5. **吸气上升**：感受全身拉伸和挺拔
6. **呼气下降**：感受全身放松

**呼吸方式**：
- **鼻吸口呼**
- 吸气约 3 秒，呼气约 5 秒
- **呼气更慢、更松、更沉**
- 把气吐干净

**要点**：
- 整个身体的重量全部向下沉
- 整个身体的关节向下放松
- 不要用下蹲的感觉让肌肉紧张
- 身体跟着呼气过程全身松弛下去

---

### 第二桩：太极混元桩

**四个核心方法**（在吸气呼气过程中完成）：

#### 方法一：虚灵顶劲

- 头顶百会穴，像有绳子把我们拎起来
- 呼气向下松时，这个"绳子"不能缩
- **不能缩脖子，头要顶起来**

#### 方法二：含胸拔背

- 含胸：把心窝（中丹田/心轮/方寸）收回去
- 拔背：把两个背往前抱
- **精神内守**：把心神收回来
- 拔背能拉伸膀胱经（阳经）

#### 方法三：沉肩坠肘

- 沉肩：肩膀松到脚心（涌泉穴）
- 坠肘：肘部曲池穴像有石头绑着向下坠
- **沉是有浮力的感觉**：像在水里，又想沉下去又有力往上升
- 手臂适度抬起，不能贴着身体（气机发动不起来）
- 也不能抬太高（气机受阻不能流通）

#### 方法四：松腰沉胯（溜臀）

- 尾骨（腰椎下面最后几节）平时是翘着的
- 站桩时要松腰沉胯，让尾骨下沉
- **不能翘屁股**，要往下沉

## 🧘 练习流程

### 从吐故纳新桩过渡到太极混元桩

1. **吐故纳新桩**：吸气上升→呼气放松（多次）
2. **含胸拔背、松腰沉胯**：做到这两个方法
3. **手适度抬起**：加入沉肩坠肘
4. **进入太极混元桩**：四个方法同时做到

### 站桩状态

- **眼睛微闭**
- **下颚微收**
- **朝着呼吸**
- **站 10-15 分钟**

### 心理调整

**可能出现的感觉**：
- 肩膀很酸
- 手很胀
- 背很硬

**心态调整**：
1. **赤子之心**：当出现酸时，不要定义它就是酸，去感受这个感受究竟是什么
2. **好奇之心**：看这个酸究竟有多酸，这个痛究竟有多痛
3. **身心分离**：心不再受身体影响，无论是肩膀酸还是背僵硬，心不要动
4. **坚定信念**：站桩是补充元气、修复身体的过程

## ⚠️ 关键要点

1. **呼吸配合**：吸气 3 秒，呼气 5 秒（呼气更长）
2. **虚灵顶劲**：头要顶起来，不能缩脖子
3. **含胸拔背**：心窝收回去，背往前抱
4. **沉肩坠肘**：肩膀松到脚心，肘部有下坠感
5. **手臂位置**：适度抬起，不贴身体也不太高
6. **松腰沉胯**：尾骨下沉，不能翘屁股
7. **心态**：赤子之心 + 好奇之心，身心分离

## ⚠️ 常见错误

| 错误 | 正确做法 |
|------|----------|
| 呼气太快 | 呼气 5 秒，更慢更松更沉 |
| 缩脖子 | 虚灵顶劲，头要顶起来 |
| 只含胸不拔背 | 含胸 + 拔背，背往前抱 |
| 手臂贴身体 | 适度抬起，让气机流通 |
| 手臂抬太高 | 适度即可，不要阻碍气机 |
| 翘屁股 | 松腰沉胯，尾骨下沉 |
| 心随身体动 | 身心分离，心不要动 |

3. **心态**：赤子之心 + 好奇之心，身心分离
4. **呼吸**：鼻吸口呼，吸气 3 秒呼气 5 秒
