## 🎯 太极拳的本质

### 核心理念

> "太极拳不是大家说的一套拳，一定是摄心归正，自省内观。"

**含义**：
- 太极拳不仅是动作练习
- 更是心性修炼
- 摄心归正，自省内观

### 心法

> "心无其心，无我之心，即是真心。真心动静，为人为意。或左或右，或出或入，这就是阴阳的变化。真气绵绵，性情依如。"

**含义**：
- 心无其心：没有杂念的心
- 无我之心：放下自我的心的状态
- 真心：真实的心
- 阴阳变化：左右、出入的变化
- 真气绵绵：气机连绵不断
- 性情依如：性情自然流露

## 🎯 正气与邪气

### 正气 = 免疫力

**正气不足的表现**：
- 免疫力低下
- 一旦遇到新的病邪，依然还会进入到原来的状态
- 固本的能力没有那么强

**正气充足的表现**：
- 免疫力强
- 病邪靠不了身
- 固本培元

---

### 邪气分类

#### 外邪

| 类型 | 表现 |
|------|------|
| **肩颈酸痛** | 肩颈部位疼痛、僵硬 |
| **三高** | 高血压、高血脂、高血糖 |
| **脾胃虚寒** | 脾胃功能弱，怕冷 |
| **虚弱** | 身体虚弱，精力不足 |
| **风寒暑湿热燥火** | 六淫外邪 |

#### 内邪

| 类型 | 表现 |
|------|------|
| **失眠** | 睡眠质量差，难以入睡 |
| **心烦气躁** | 心情烦躁，容易生气 |
| **抑郁情绪** | 情绪低落，消极悲观 |
| **喜怒哀乐悲思恐** | 七情内伤 |

## ❓ 常见问题解答

### Q1：什么是太极拳的本质？

**答**：太极拳不是大家说的一套拳，一定是摄心归正，自省内观。太极拳不仅是动作练习，更是心性修炼。

### Q2：什么是正气？

**答**：正气 = 免疫力。正气充足则免疫力强，病邪靠不了身；正气不足则免疫力低下，容易生病。

### Q3：什么是外邪？

**答**：外邪是指肩颈酸痛、三高、脾胃虚寒、虚弱、风寒暑湿热燥火等外部病邪。

### Q4：什么是内邪？

**答**：内邪是指失眠、心烦气躁、抑郁情绪、喜怒哀乐悲思恐等内部情绪问题。

### Q5：太极拳如何提升正气？

**答**：太极拳通过练习充盈正气，增强免疫力，固本培元，驱除外邪，调理内邪。

### Q6：太极拳对焦虑有帮助吗？

**答**：有帮助。太极拳可以驯服潜意识，情绪稳定，对焦虑有改善作用。

### Q7：太极拳对睡眠有帮助吗？

**答**：有帮助。太极拳可以精神内守，静坐课程帮助入睡，对睡眠有改善作用。

### Q8：太极拳要练多久才能提升正气？

**答**：太极拳非一日之功，是量变到质变的过程。建议每天练习，坚持一段时间后会感受到正气充盈，免疫力提升的效果。
