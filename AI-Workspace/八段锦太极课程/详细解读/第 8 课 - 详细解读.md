## 📌 课程主题

**双手攀足固肾腰**是八段锦的收官动作，通过揉腰、存气屏息、推胆经、搬跟腱、仰头快呼快吸等组合动作，**固肾腰、强筋骨、治疗腰肌劳损和手脚麻木**，达到身心齐修、收获喜悦和幸福的养生境界。

**核心原理**：肾主藏，是养生的根本。五脏六腑之精华皆由肾所藏，是人青春永驻的资本。肾精充盈时，五脏六腑都得到较好的生存环境。

## 🎯 动作要领（分步骤详解）

### 一、起势准备

| 步骤 | 动作描述 | 呼吸 | 关键要点 |
|------|----------|------|----------|
| 1 | 双手展平，从身体两侧向上抬起 | 吸气 | 手臂伸直 |
| 2 | 双手至中丹田（胸口位置） | 吸气 | 气吸满 |
| 3 | 手扇形打开，三贴合 | - | 掌根贴、手臂贴、身体贴合 |
| 4 | 推肩展臂 | - | 双臂伸直 |
| 5 | 下蹲半蹲，重心右移 | - | 虚膝下蹲 |
| 6 | 左脚跨开半步 | - | 与肩同宽 |
| 7 | 手移至腰间 | - | 准备揉腰 |

### 二、核心动作（3 次循环）

#### 第一步：揉腰（9 次）

| 要点 | 说明 |
|------|------|
| **动作** | 用掌根按压肾脏位置，快速上下搓 |
| **次数** | 9 次 |
| **目的** | 以搓热为主，激活肾脏区域 |
| **呼吸** | 自然呼吸 |

#### 第二步：存气屏息（降火烧麒轮）⭐ 关键

| 要点 | 说明 |
|------|------|
| **动作** | 吸气，气吸满 |
| **停顿** | **略微停顿 3-5 秒**（关键！容易忘记！） |
| **意念** | 想双手发热，敷热腰间 |
| **名称** | 这个动作叫"降火烧麒轮" |
| **呼吸** | 吸气→停顿 3-5 秒 |

**降火烧麒轮详解**：
- **降火**：将心火下降到肾水
- **烧麒轮**：麒轮指肾脏区域，温热肾脏
- **作用**：心肾相交，温补肾阳
- **要点**：吸气后停顿 3-5 秒，想双手发热

#### 第三步：推胆经（呼气）

| 要点 | 说明 |
|------|------|
| **动作** | 呼气，手顺着大腿外侧向下推 |
| **路线** | 推胆经，一直到手到脚面 |
| **要点** | 手碰不到脚面也要尽量往下 |
| **呼吸** | 呼气 |

**胆经路线**：
- 从腰部沿大腿外侧向下
- 经过膝盖外侧
- 到脚面

#### 第四步：搬住跟腱（关键动作）⭐

| 要点 | 说明 |
|------|------|
| **动作** | 手掌根不动，手指内扣 |
| **目标** | **搬住跟腱**（脚跟上面的大筋） |
| **要点** | 腿弯曲也要扣到脚腕 |
| **关键** | 手掌根不动，手指内扣 |

**搬住跟腱详解**：
- **跟腱**：脚跟上面的大筋
- **动作**：手指内扣，扣住跟腱
- **要点**：手掌根不动，手指内扣
- **常见错误**：搬不住跟腱，手滑动

#### 第五步：吸气仰头 + 快呼快吸（3 次）

| 步骤 | 动作 | 呼吸 | 要点 |
|------|------|------|------|
| 1 | **吸气仰头** | 吸气 | **二目圆睁** |
| 2 | 拉伸后背督脉的筋 | - | 拉伸督脉 |
| 3 | **快呼快吸 3 次** | 快呼快吸 | 身体往回勾 |
| 4 | 身体往回勾 | - | 身体收缩 |

**要点**：
- **吸气仰头**：二目圆睁，拉伸后背督脉
- **快呼快吸**：3 次快速呼吸
- **身体往回勾**：身体收缩，不是伸直

#### 第六步：起身（从内侧）⭐

| 步骤 | 动作 | 呼吸 | 要点 |
|------|------|------|------|
| 1 | 手从内侧慢慢起来 | 吸气 | 从肝经（大腿内侧） |
| 2 | 到大腿内侧→腹股沟 | - | 沿肝经上行 |
| 3 | 到腹股沟后，移动到**带脉** | - | 带脉排毒 |
| 4 | 从带脉向后旋转到腰间 | - | 回到腰间 |
| 5 | 回到起点 | - | 准备下一次循环 |

**从内侧起身详解**：
- **路线**：肝经（大腿内侧）→腹股沟→带脉→腰间
- **作用**：避免眩晕，慢慢起来
- **带脉排毒**：手到带脉时舒展打开，排出毒素
- **常见错误**：起身时手从外侧起，导致眩晕

### 三、收势（第 3 次循环后）

| 步骤 | 动作 | 呼吸 | 要点 |
|------|------|------|------|
| 1 | 第 3 次做完后，手不要往后移动 | - | 与第 1、2 次不同 |
| 2 | 手顺着身体到头顶 | 吸气 | 手臂上举 |
| 3 | 双臂展开 | - | 手臂展开 |
| 4 | 并脚，五指张开 | - | 并脚站立 |
| 5 | 向外展开身体 | - | 身体展开 |
| 6 | 旋掌，吐气向前 | 呼气 | 旋掌向前 |
| 7 | 吸气收回 | 吸气 | 回到中丹田 |
| 8 | 呼气下沉 | 呼气 | 回到静站 |

**收势特别注意**：
- **第 3 次循环后**：手不要往后移动到腰间
- **直接起身**：手顺着身体到头顶，展开收势
- **常见错误**：第 3 次循环后手继续往后，再做一次

### 四、呼吸配合

| 动作 | 呼吸 | 说明 |
|------|------|------|
| 揉腰 9 次 | 自然呼吸 | 快速搓 |
| 存气屏息 | **吸气→停顿 3-5 秒** | 降火烧麒轮 |
| 推胆经 | **呼气** | 一直到手到脚面 |
| 搬住跟腱 | - | 手内扣 |
| 吸气仰头 | **吸气** | 二目圆睁 |
| 快呼快吸 | **快呼快吸 3 次** | 身体往回勾 |
| 起身 | **吸气** | 从内侧慢慢起来 |
| 回到起点 | 自然呼吸 | 准备下一次循环 |

### 五、完整动作流程

```
起势 → 
第 1 次循环：揉腰 9 次→存气屏息→推胆经→搬跟腱→吸气仰头快呼快吸 3 次→从内侧起身 → 
第 2 次循环：同上 → 
第 3 次循环：同上 → 
收势（手不顺腰，直接上举展开）
```

## ❓ 常见问题解答

### Q1：什么是"降火烧麒轮"？

**答**：降火烧麒轮是存气屏息动作的名称。
- **降火**：将心火下降到肾水
- **烧**：温热、温暖
- **麒轮**：指肾脏区域（中医术语）
- **动作**：吸气后停顿 3-5 秒，想双手发热，敷热腰间
- **作用**：心肾相交，温补肾阳
- **要点**：吸气后停顿 3-5 秒（容易忘记！）

### Q2：为什么要存气屏息 3-5 秒？

**答**：存气屏息是双手攀足固肾腰的关键动作。
- **原因**：
  1. 气吸满后停顿，增强气的运行
  2. 想双手发热，敷热腰间，温补肾阳
  3. 降火烧麒轮，心肾相交
- **时间**：3-5 秒，不要太长
- **常见错误**：忘记存气屏息动作

### Q3：搬住跟腱具体怎么做？

**答**：搬住跟腱是手部的关键动作。
- **跟腱**：脚跟上面的大筋
- **动作**：手掌根不动，手指内扣
- **目标**：扣住跟腱
- **要点**：
  1. 手掌根不动，固定位置
  2. 手指内扣，扣住跟腱
  3. 腿弯曲也要扣到脚腕
- **常见错误**：搬不住跟腱，手滑动

### Q4：为什么要从内侧起身？

**答**：从内侧起身是为了避免眩晕。
- **路线**：肝经（大腿内侧）→腹股沟→带脉→腰间
- **作用**：
  1. 避免快速起身导致眩晕
  2. 拉伸肝经，疏肝解郁
  3. 带脉排毒，排出毒素
- **常见错误**：起身时手从外侧起，导致眩晕

### Q5：带脉排毒是什么意思？

**答**：带脉排毒是起身时的动作。
- **带脉**：环绕腰部的经络
- **排毒**：排出腰部毒素
- **动作**：手到带脉时舒展打开
- **作用**：
  1. 排出腰部毒素
  2. 改善腰部肥胖
  3. 疏通带脉经络
- **要点**：手到带脉时舒展打开

### Q6：第 3 次循环后为什么手不往后移动？

**答**：第 3 次循环后直接收势。
- **第 1、2 次**：循环后手回到腰间，准备下一次
- **第 3 次**：循环后手不往后，直接上举收势
- **原因**：完成 3 次循环，进入收势
- **常见错误**：第 3 次循环后手继续往后，再做一次

### Q7：这个动作为什么能固肾腰？

**答**：双手攀足固肾腰有多重固肾腰机制。
1. **揉腰**：激活肾脏区域，促进肾脏血液循环
2. **存气屏息**：降火烧麒轮，温补肾阳
3. **推胆经**：疏通经络，增强气血运行
4. **搬跟腱**：拉伸肾经，强筋健骨
5. **吸气仰头**：拉伸督脉，生发阳气
6. **从内侧起身**：拉伸肝经，疏肝补肾

### Q8：为什么肾为先天之本，脾为后天之本？

**答**：这是中医的核心理论。
- **肾为先天之本**：
  - 肾藏精，是生命的根本
  - 五脏六腑之精华皆由肾所藏
  - 是父母遗传的生命能量
- **脾为后天之本**：
  - 脾主运化，消化吸收营养
  - 是后天获得的营养来源
  - 滋养全身
- **关系**：
  - 先天（肾）滋养后天（脾）
  - 后天（脾）补充先天（肾）
  - 固肾腰有助于脾胃功能恢复

## 📋 练习计划建议

### 第一周：动作学习期

| 时间 | 练习内容 | 目标 |
|------|----------|------|
| 第 1-2 天 | 学习起势和揉腰 | 掌握揉腰 9 次、存气屏息 |

| 时间 | 练习内容 | 目标 |
|------|----------|------|
| 第 8-10 天 | 完整动作练习（慢速） | 3 次循环连贯，呼吸配合 |
| 第 11-14 天 | 完整动作练习（正常速度） | 3 次循环 + 收势 |

### 长期练习

- **晨练**：作为八段锦第八式，3 次循环 + 收势
- **腰酸时**：简化版（只揉腰 + 存气屏息）3 次，缓解腰痛
- **睡前**：缓慢练习 1 次循环，帮助入睡
