# AI时代的职业图谱

> 讲者：熊辉（香港科技大学广州）

## 智能的四个境界

| 境界 | 层次 | AI现状 |
|------|------|--------|
| 1. 博闻强记 | 知识记忆 | ✅ AI已完全超越人类 |
| 2. 触类旁通 | 跨领域应用 | ✅ AI远远超越人类 |
| 3. 一叶知秋 | 预测与推理 | ⚠️ AI快速进步中 |
| 4. 无中生有 | 0→1创新 | ❌ AI目前做不到 |

## 核心洞察

人类在AI时代的核心竞争力在于**无中生有的创新能力**。
