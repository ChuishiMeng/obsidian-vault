# Amazon Kiro IDE

## 基本信息

- **公司**：Amazon
- **产品**：Kiro IDE
- **定位**：AI-powered IDE
- **发布时间**：2026年4月
- **状态**：新产品，与 Cursor、VS Code Copilot 竞争

## 核心功能

### 1. AI-native 开发环境

- 内置 AI 助手（非插件形式）
- 支持多模型切换
- 上下文感知代码补全

### 2. 与 Amazon 生态集成

- AWS 服务深度集成
- Lambda 函数智能生成
- CloudFormation 模板辅助

### 3. 开发者体验

- 界面类似 Cursor
- 支持 Spec Mode（先定义规格再生成）
- 实时代码审查

## 行业影响

### 对 Cursor 的冲击

- Cursor 年化收入 $2B，Kiro 直接竞争
- Amazon 有 AWS 生态优势
- 可能挤压 Cursor 在企业市场份额

### 对 OpenClaw 用户的意义

1. **多 IDE 选择**：不再局限于 Cursor + Claude Code
2. **AWS 场景优化**：如果孟老师团队使用 AWS，Kiro 可能更合适
3. **价格竞争**：Cursor $20/月，Kiro 可能有更优惠的企业定价
