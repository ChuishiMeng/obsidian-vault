# Everything Claude Code (ECC) 完整指南

> 110K+ Stars 项目，黑客松霍圣者历时10个月打磨

## ECC 是什么

装在 `~/.claude` 目录下的Claude Code插件集合。

比喻：原版Claude Code = 聪明但无约束的实习生；ECC = 配了公司制度和操作手册的实习生。

## 六大核心模块

| 模块 | 数量 | 说明 |
|------|------|------|
| Commands | 60+ | `/` 触发的命令 |
| Agent Strategy | 28个 | 专用AI子代理 |
| Hooks | 8种 | 事件驱动自动化 |
| Skills | ~125个 | 领域知识库 |
| Rules | - | 全局编码规范 |
| MCP | - | 外部服务集成 |
