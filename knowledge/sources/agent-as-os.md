# 我把 Agent 当操作系统用了一天

## 核心体验

### Agent 不应是工具窗口，而应是工作世界

**问题**：过去把Agent当工具窗口，有事找它，没事不来往。

**实验**：全屏打开Agent，假装今天切不出去。

**发现**：当一天的目标、调研、思考都在同一条对话流里发生，工作开始变得连贯——信息被连接、任务被推进。

### TOP 3 任务管理
从备忘录到作战计划，Agent成为工作中心。
