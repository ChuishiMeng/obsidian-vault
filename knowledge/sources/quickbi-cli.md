# Quick BI CLI 开放：数据操作迈入 AI 原生时代

> 来源：瓴羊 Quick BI 和 Dataphin 公众号 | 日期：2026-04-07

## 核心变化

Quick BI 把开放平台的 **134 项 API 功能**全部转化为终端命令，开发者（或 AI Agent）可以用一行自然语言完成原来需要写大量代码的操作。

## 四个高频场景

| 场景 | 原来 | 现在 |
|------|------|------|
| 报表外嵌 | 写代码生成 Ticket + 定时续期，跨天开发 | 一句话秒级完成 |
| 数据资产盘点 | 手动翻页面或写复杂 API | 一句话获取 Top10 |
| 用户/权限批量管理 | 后台逐一操作 | 一句话完成 |
| 问数 Skill 配合 | 批量赋权流程繁琐 | CLI 赋权 → Skill 问数一条龙 |

## AI Agent 集成

支持 OpenClaw、Qoder work、Claude Code、悟空等主流 Agent，提供专门 AI 安装指南。
