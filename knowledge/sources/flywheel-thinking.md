# 飞轮思维

> 核心问题：每天做的事，有没有让明天变得更好？

## 核心概念

- **飞轮**：今天的努力让明天加速（积累型）
- **单点消耗**：今天做完明天重新开始（无积累）

## Anthropic 飞轮案例

选择 Coding 不是因为市场大，而是做 Coding 本身加速研发：
- Claude Code → 更好的模型 → 更好的工具 → 再加速研发
- 护城河 = 自我迭代的结构本身

## Token 飞轮

AI 的本质是 Token 换时间。关键：花出去的 Token 能否带回更多价值？
- 花 10 亿 → 变成 11 亿（10% 回报）→ 闭环可转
- 如果只是处理信息，价值未实现 → 飞轮停转

## 写作的四个标准

| 标准 | 类型 |
|------|------|
| 今天有阅读量 | 短期 |
| 今天有物质回报 | 短期 |
| 提升自己的认知 | 长期 ⭐ |
| 十年后还愿意收录进书里 | 长期 ⭐ |

## 关键推论

不是「要做什么」，而是「要停止做什么」。80% 精力投在 10%-20% 的核心事情上。
