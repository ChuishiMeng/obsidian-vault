# Vibe Coding

## 概念定义

**Vibe Coding**是一种AI辅助编程的方法论，强调在AI帮助下快速原型开发，通过自然语言描述需求，让AI生成代码，开发者专注于整体架构和关键决策。

## 核心要点

1. **快速原型** — 用AI快速搭建可运行的原型
2. **自然语言驱动** — 用自然语言描述需求而非手写代码
3. **迭代优化** — 通过多轮对话逐步完善
4. **人类把关** — 开发者负责架构和关键决策

## 与AI编程的区别

- **AI编程** — 广义概念，包括所有AI辅助编程
- **Vibe Coding** — 特定的方法论，强调快速和氛围
