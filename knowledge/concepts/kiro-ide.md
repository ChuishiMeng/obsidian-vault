# Kiro IDE

Amazon 于 2026 年 4 月推出的 AI-first IDE，核心特色是双模式设计。

## 核心特性

| 模式 | 说明 |
|------|------|
| **Spec Mode** | 先写规格说明，AI 根据规格自动生成代码 |
| **Vibe Mode** | 自然语言描述需求，AI 全自动生成（Vibe Coding） |

## 行业意义

- 标志着 Vibe Coding 从极客文化进入大厂正式产品线
- Harvard 和 Google 同时介入 Vibe Coding 研究
- 92% 开发者已采用 AI 辅助编程，生产力提升 3-5x
- Karpathy 提出"Agentic Engineering"替代"Vibe Coding"作为专业术语
