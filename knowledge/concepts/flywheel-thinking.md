# 飞轮思维 Flywheel Thinking

## 定义

一种判断事情价值的方法论：今天的努力是否让明天加速？如果是，则属于飞轮的一部分；如果不是，则是单点消耗。

## 核心要素

1. **积累效应**：每次投入都在前一次的基础上叠加
2. **自我加速**：飞轮转得越快，下一次转动越省力
3. **结构化护城河**：不是单一优势，而是自我迭代的结构本身

## 经典案例

- **Anthropic**：用 Claude Code 做研发 → 更好的模型 → 更好的工具 → 循环加速
- **Token 飞轮**：花 Token → 产出价值 → 价值换钱 → 买更多 Token
- **写作飞轮**：每次写作提升认知，认知提升反过来提高写作质量

## 判断标准

> 你现在做的事情，是飞轮的一部分，还是单点消耗？
