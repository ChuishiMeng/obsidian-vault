# AI 编程 (AI Coding)

## 概念定义

**AI编程**是使用AI工具（如Claude Code、Copilot等）辅助软件开发的过程，包括代码生成、审查、重构、测试、文档编写等。

## 核心要点

### AI编程工具

- **Claude Code** — Anthropic的命令行AI编程助手
- **GitHub Copilot** — GitHub的代码补全工具
- **ECC (Everything Claude Code)** — 110K+ Stars的Claude Code插件集合

### ECC六大模块

| 模块 | 数量 | 说明 |
|------|------|------|
| Commands | 60+ | `/`触发的命令 |
| Agent Strategy | 28个 | 专用AI子代理 |
| Hooks | 8种 | 事件驱动自动化 |
| Skills | ~125个 | 领域知识库 |
| Rules | - | 全局编码规范 |
| MCP | - | 外部服务集成 |

## 使用建议

- **先想清楚方向** — 不要用AI更快地维持现状
- **代码审查必须** — 测试通过≠代码正确
- **分步骤执行** — 避免一次做太多导致超时
