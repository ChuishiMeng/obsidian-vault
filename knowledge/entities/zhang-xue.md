# 张雪

**春风动力（CFMOTO）创始人，从县城修车工到营收7亿车企老板**

## 简介

张雪是春风动力创始人，19岁开二手摩托车追记者3小时只为上电视，2013年到重庆创业，2025年年营收7亿，带领国产车队征战达喀尔拉力赛。

## 关键经历

| 时间 | 事件 |
|------|------|
| 2006 | 19岁，追记者3小时只为上电视 |
| 2013 | 到重庆创业 |
| 2017 | 创立春风机车，第一年卖800台 |
| 2018 | 决定自研发动机（公司营收仅3千万）|
| 2023 | 带领国产车队征战达喀尔拉力赛 |
| 2025 | 年营收7亿 |

## 核心品质

- **极致行动力**：想做就做
- **长期主义**：愿意做漫长但关键的投资
- **不怕失败**：多次失败仍坚持
