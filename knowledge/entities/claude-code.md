# Claude Code

**Anthropic推出的AI编程助手，命令行界面的代码协作工具**

## 简介

Claude Code是Anthropic推出的AI编程助手，通过命令行界面与开发者协作，支持代码审查、重构、测试、文档生成等功能。

## 核心功能

- 代码审查与问题发现
- 自动重构与修复
- 测试生成与验证
- 文档编写
- 项目级理解

## 生态系统

- **ECC (Everything Claude Code)** — 110K+ Stars的插件集合，提供60+命令、28个Agent、~125个Skills
