# OpenClaw

**2026年最火爆的AI Agent开源项目，GitHub史上增长最快仓库**

## 简介

OpenClaw是一个AI Agent操作系统，定义全新一代Agent的必备基础功能：易用性、高度普适性、可定制化功能、自主功能进化。

## 八大核心模块

| 模块 | 功能 |
|------|------|
| Gateway | 网关，管理渠道连接、路由、会话 |
| Channel | 消息平台集成（飞书、企微、微信等）|
| Session | 保存长短期对话、记忆 |
| Agent | 处理具体任务 |
| Tools | 工具接入 |
| Node | 节点管理 |
| Sandbox | 沙箱环境 |
| Skills | 技能系统 |

## 三大核心特性

1. **无限记忆** — 长短期记忆管理
2. **自主进化** — 自我迭代改进
3. **灵活载入技能** — 自由组装Skills
