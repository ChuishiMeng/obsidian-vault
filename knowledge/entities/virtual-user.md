# 虚拟用户（Virtual User / Synthetic Respondent）

## 基本信息

| 属性 | 值 |
|------|------|
| **类型** | 研究领域 / 技术概念 |
| **核心场景** | 电商平台市场调研 |
| **所属项目** | 京东虚拟消费者 |
| **目标会议** | KDD |
| **文献规模** | 60 篇检索，15 篇深度解读 |

## 核心思路

基于电商行为数据（浏览、搜索、购买、评价）构建虚拟用户 Persona，模拟市场调研问卷响应，替代传统问卷调查。

## 应用场景

| 场景 | 传统方法 | 虚拟用户方法 | 价值 |
|------|---------|------------|------|
| 新品定价 | 招募问卷 4 周 | 分钟级生成 | 成本降 90% |
| 流失分析 | 电话回访 <10% 接通 | 虚拟流失用户 | 全量覆盖 |
| 文案测试 | A/B 测试影响转化 | 虚拟反馈筛选 | 无副作用 |

## 核心挑战

1. **态度矛盾**：同一虚拟用户回答不一致（>40%，P11 Lost in Simulation）
2. **回归均值**：倾向友好/社会期望回答
3. **分布偏差**：系统性偏离真实人群
4. **评估困难**：无完美 baseline

## 本研究方案

- 态度因子建模 + 一致性约束（核心创新）
- 电商行为 → 态度推断 → Persona 生成
- 三维评估：态度一致性、分布相似度、预测准确率

## 京东独有优势

海量行为数据 + 丰富用户画像 + 真实调研需求 + 验证环境
