# 《指数基金投资指南》整体解读

> 作者：银行螺丝钉 | 投资小新深度解读

## 核心方法论

**一句话总结**：在盈利收益率大于10%时定投低估指数基金，小于6.4%时卖出，长期坚持可获得15-20%年复合收益率。

## 三大核心理念

1. **财务自由** = 手握大量能"钱生钱"的资产 + 充足时间做自己想做的事
2. **指数基金优势**：长生不老、长期上涨、成本低
3. **定投精髓**：定期定额，分散成本，不必精准判断高低点

## 估值方法体系

### 盈利收益率法（格雷厄姆）
| 盈利收益率 | 操作 |
|------------|------|
| >10% | 分批定投 |
| 6.4%~10% | 坚定持有 |
| <6.4% | 分批卖出 |

### 博格公式法
> 年复合收益率 = 股息率 + 市盈率变化率 + 盈利增长率
