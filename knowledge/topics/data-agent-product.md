# 智能问数产品（JoyAnalysis）

## 概述

智能研判 JoyAnalysis 是面向业务人员的智能数据问数与分析助手，将数据获取时间从天级缩短到分钟级。产品定位为「问即答，答即准」的企业级 ChatBI 系统，解决业务人员不会 SQL、等待数据团队、数据口径不一致三大痛点。基于京东内部 DataAgent 项目的技术积累，服务于营销中台业务场景。

## 核心功能

| 功能 | 说明 |
|------|------|
| **自然语言问数** | 用自然语言查询业务数据，支持多模态私域数据整合 |
| **可视化展示** | 自动生成图表，让数据展示更直观清晰 |
| **意图澄清** | 问询不明确时通过反问澄清用户意图 |
| **辅助决策** | 信息总结、趋势分析、异常分析、归因分析、分析建议 |
| **空间隔离** | 按业务空间切换数据范围，支持公共数据和个人数据 |
| **对话管理** | 历史对话保存、修改、删除，支持多轮对话 |
| **数据模型浏览** | 查看可用数据模型、字段信息、业务术语 |

## 技术架构

- **NL2SQL 引擎**：基于大模型（GPT-4、Claude 3.5 等）将自然语言转换为 SQL
- **数据治理**：支持结构化和非结构化数据的敏捷整合、治理、归类
- **持续迭代**：基于交互反馈持续优化查询和分析能力

## 成功指标

| 指标 | 目标值 |
|------|--------|
| SQL 生成准确率 | ≥ 85%（简单 ≥ 90%，复杂 ≥ 75%）|
| 查询响应时间 | ≤ 40 秒 |
| 用户满意度 | ≥ 4.5/5 |
| 周活跃用户 | ≥ 500 人 |
| 数据团队提效 | ≥ 50% |
