# 乐高智能化

## 概述

乐高智能化是在京东营销中台「乐高 3.0」页面搭建系统基础上，增加 AI 能力实现「一句话生成页面」的升级项目。对标 Figma AI、v0.dev、bolt.new 等产品，通过自然语言控制乐高组件，大幅降低页面搭建门槛。核心创新是**组件控制协议（CCP）**——LLM 不直接操作系统，而是输出标准化操作指令，由执行引擎翻译为乐高 API 调用。

## 核心功能

| 功能 | 说明 | 优先级 |
|------|------|--------|
| **一句话生成页面** | 自然语言描述生成完整页面结构，支持 2-3 个方案选择 | P0 |
| **模板智能推荐** | 基于页面描述推荐合适的楼层模板 | P0 |
| **素材智能填充** | AI 自动为坑位匹配素材内容 | P1 |
| **内容智能生成** | 自动生成文案、标题等文本内容 | P1 |
| **合规智能检查** | AI 审核页面内容合规性 | P1 |
| **对话式配置** | 通过自然语言对话完成页面配置 | P2 |

## 技术架构

```
用户交互层（对话输入框、预览画布、调整面板）
    ↓
AI 服务层（意图理解 → 结构规划 → 内容推荐）
    ↓
组件控制协议层（CCP：LLM 输出标准化操作指令）
    ↓
执行引擎层（指令解析 → 规则校验 → API 翻译）
    ↓
乐高核心层（模板管理、页面管理、素材管理、渲染引擎）
```

**关键概念映射**：页面=画布、楼层=Frame、模板=Component、坑位=Element、素材=Content
