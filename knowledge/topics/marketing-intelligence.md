# 营销智能化

## 概述

营销智能化是京东营销中台的 AI 策略生成系统，通过自然语言输入活动需求，自动完成客群分析、营销策略生成和执行配置。核心能力是将运营人员输入的活动主题、周期、目标、商品范围等自然语言描述，经过五层深度分析（客群画像 → 行为转化 → 触点偏好 → 响应因素 → 策略生成），输出可执行的营销策略并对接 MA 系统部署。

## 核心功能

| 功能 | 说明 |
|------|------|
| **自然语言活动输入** | 输入活动主题、周期、商品范围、目标等，系统自动解析 |
| **五层客群分析** | 客群画像 → 全链路行为 → 触点偏好 → 决策驱动因素 → 营销响应规律 |
| **策略自动生成** | 基于分析结果生成包含权益、渠道、话术的完整营销策略 |
| **MA 系统对接** | 策略直接部署到营销自动化系统 |
| **A/B 测试配置** | 支持策略的 A/B 测试验证 |
| **差异分析与调优** | 预期 vs 实际结果对比，根因分析，下一轮优化建议 |

## 技术架构

```
用户输入层（自然语言活动需求）
    ↓
大模型工作流编排层（意图识别 → 任务分解 → 数据路由 → 执行编排）
    ↓
数据查询层（SQL/DSL 生成、数据分析引擎、外部 API）
    ↓
分析结果聚合层（语义化解读与洞察提炼）
    ↓
营销策略生成层（基于分析结果生成可执行策略）
    ↓
策略输出与执行层（MA 系统对接、人工审核、A/B 测试）
```

**五层分析模型**：
1. **客群基础属性画像** — 人口属性、消费能力、品牌设备特征、显著性差异
2. **全链路行为与转化潜力** — 浏览意向、流失节点定位、历史转化、场景化刚需
3. **触点偏好与交互响应** — 渠道偏好、时机频次、最优路径、内容形式偏好
4. **营销响应与决策驱动** — 历史活动响应、核心决策点、转化顾虑、内容偏好
5. **策略生成与执行配置** — 权益设计、渠道策略、话术生成、执行排期
